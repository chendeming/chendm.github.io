openclaw Multi-Agent and multi-Bot
I was hoping that an openclaw would allow him to do multiple things separately, instead of doing everything in one session (and only serially). How do you do that? Watching a teaching video, there are two ways, one is to create multiple agents, and the other is to create multiple bots.
- openclaw can generate multiple agents in the main session after the command line, so that these agents have their own personality and do different things. The blogger creates multiple group sessions in the fly book, and gives the session ID to the agent, and the agent can answer questions in the session.
